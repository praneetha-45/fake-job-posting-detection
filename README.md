Fake Job Posting Detection System

An AI-powered machine learning application that detects fraudulent and legitimate job postings using Natural Language Processing (NLP) and predictive modeling techniques.

Features
Detects fake and real job postings in real time
NLP-based text preprocessing and analysis
TF-IDF vectorization for feature extraction
Logistic Regression-based classification model
Interactive Streamlit web interface
Predictive fraud scoring system
Tech Stack
Python
NLP (Natural Language Processing)
Scikit-learn
TF-IDF Vectorization
Logistic Regression
Streamlit
Pandas
NumPy
Project Workflow
Data collection and preprocessing
Text cleaning and stopword removal
TF-IDF feature extraction
Model training using Logistic Regression
Fraud prediction and scoring
Streamlit-based deployment
Dataset

Dataset used: Fake Job Postings Dataset from Kaggle.

How to Run the Project
Install Dependencies
pip install -r requirements.txt
Run Streamlit App
streamlit run app.py
